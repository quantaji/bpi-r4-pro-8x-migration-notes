# Legacy Semantic Tagging Archive

Archived on 2026-06-08.

This directory contains the old semantic tagging workflow:

```text
tag_rules.md
patch_tagging_skill.md
```

These files are not the entry point for the current Project Phase 1a workflow.
Current Project Phase 1a output stores only feature routing tags and is driven by:

```text
feature_routing_skill.md
rules/feature-tags-v1.json
rules/feature-seed-rules-v1.json
scripts/phase-1-feature-route.py
```

The archived files are kept because their `origin`, `scope`, and semantic
modification concepts may still be useful during later feature-cluster review
and migration design. They should not be used for the mechanical per-file
feature-routing seed pass.
